# 더 비쌀까 쌀까?

음식 관련 컨텐츠의 인기와 타임킬링 컨텐츠의 성장이라는 배경을 통해 개발하게 된 타임킬링용 컨텐츠

주어진 두 음식에 대해 주어진 음식의 가격을 비교하며 타임 킬링하는 동시에 사용하는 유저들 간에 랭킹 시스템을 통해 경쟁도 가능하도록 함

퀴즈에 올라온 음식 사진의 설명과 가격을 보며 사용자가 전국의 맛집 정보를 동시에 파악할 수 있도록 함

## 멤버

Back-end: 공준형, 김유민

Front-end: 배병욱

Database: 이동제
